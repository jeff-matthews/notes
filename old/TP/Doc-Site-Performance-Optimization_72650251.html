Goal

Ensure that the AffiniPay developer site leverages web development best practices to optimize page speed. 

Here's great article on optimizing Jekyll performance using Gulp: http://savaslabs.com/2016/10/19/optimizing-jekyll-with-gulp.html

TitleUser StoryImportanceNotes
TP-223Enable site resource compressionCompressing resources with gzip or deflate can reduce the number of bytes sent over the network.Must have


TP-224Eliminate render-blocking JS/CSS in above-the-fold site content

Prevent CSS file from delaying the initial render for the end user by inserting CSS that’s critical to above-the-fold content into the HTML head. By “above the fold” I mean content viewable to the user on the initial page load before interacting with the site.

Execute JS asynchronously, meaning it will be executed when it’s ready while the page is loading without interfering with the render.

Must haveSee the reference article for an example.
TP-225Leverage browser cachingSetting an expiry date or a maximum age in the HTTP headers for static resources instructs the browser to load previously downloaded resources from local disk rather than over the network.Must have
TP-226Optimize images

Manually shrink image file sizes, then set up gulp to optimize them even further for the smallest images possible.

Must haveI've already added a working Gulp task for this this to the gulpfile (minify-images) as part of TP-31, but it may not be complete. Verify working example against reference article.

TP-227

Minify HTML

Compacting HTML code, including any inline JavaScript and CSS contained in it, can save many bytes of data and speed up download and parse times.

Must haveI've already added a working Gulp task for this this to the gulpfile (minify-html) as part of TP-31.
TP-228Replace CSS/JS linksUpdate the <script> and <link> tags in all page layouts to point to the minified and concatenated files (e.g., styles.css and scripts.js).Must haveWe'll also need to exclude or delete all css and js files being copied to the _site directory except for the concatenated files.
TP-121Mobile optimizationTune the site to address lag/choppiness on mobile devicesMust haveQA opened this issue a while back. Double-check to ensure it's still relevant.
