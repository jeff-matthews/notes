Tech Pubs : Anchor links in jekyll

Created by Jeff Matthews, last modified on Feb 01, 2017

Problem

Sometimes, especially with API reference, we need to create anchor links for headings that contain special characters. For example:

<h4>merchant.provisioned</h4>
<a href="#merchant.provisioned>merchant.provisioned</a>

When Jekyll compiles that code in the markdown file, the link won't work because it contains a special character (.).Â 

Solution

remove the special character from the link URL; not the link label. For example:

<h4>merchant.provisioned</h4>
<a href="#merchantprovisioned>merchant.provisioned</a>
