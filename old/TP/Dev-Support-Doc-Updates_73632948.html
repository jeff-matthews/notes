Tech Pubs : Dev Support Doc Updates

Created by Jeff Matthews, last modified on Apr 10, 2017

Dev support issues provide the best inspiration for doc site updates because they come straight from our primary audience–developers. I created the original developer FAQ after surveying a small sample size of devsupport@affinipay.com emails between 09/26/2016 and 11/01/2016 (TP-113). Emails to devsupport@affinipay.com no longer go to a distribution list of recipients like they used to; they go to Spencer Kees so he can triage and drive issues to resolution.

We're currently in the process of determining the most efficient means of tracking dev support issues across teams to ensure we're providing first-rate customer service and that all stakeholders have visibility into issue progress. We should continue monitoring dev support issues so we can add trending questions to the FAQ or add/update other doc site topics as necessary. This document provides some general guidance on vetting issues for inclusion in the doc site.
